# AutoTest Recorder（Chrome 插件）

## 功能

- 在网页中 **点击元素** 会记录为一步（域名 + CSS 选择器 + 动作）。
- 在 `input/textarea/contenteditable` 中 **输入文本** 会记录为一步（包含输入值）。
- 在插件弹窗点击 **开始测试** 会按列表顺序回放：点击 / 输入。
- 如果某一步 **找不到元素**，会在弹窗显示错误（第几步失败）。

## 安装与使用

1. 打开 Chrome，进入 `chrome://extensions/`
2. 打开右上角 **开发者模式**
3. 点击 **加载已解压的扩展程序**，选择本目录 `autotest-recorder-extension`
4. 打开你要测试的网页
5. 点击扩展图标：
   - 点 **开始录制** → 回到网页进行点击/输入
   - 点 **停止录制**
   - 点 **开始测试** 执行回放

## 数据

步骤保存在 `chrome.storage.local`，支持 **导出/导入 JSON**。
